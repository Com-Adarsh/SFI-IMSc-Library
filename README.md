# Abhimanyu Learning Space

A focused academic environment for CUSAT students featuring a professional digital library.

## Features

### Professional Library
- **9 Subjects** across IMSC curriculum
- **10 Semesters** organized content
- **Question Papers, Textbooks & Notes**
- **Moderated Upload System** with email confirmation

### Secure Upload System
- **Moderated Reviews** before publication
- **Email Notifications** for contributors
- **PDF-only** with 50MB limit
- **Drag & Drop** interface

## Quick Start

### Prerequisites
- Node.js 18+
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/Com-Adarsh/SFI-IMSc-Library.git
cd SFI-IMSc-Library

# Install dependencies
npm install

# Run development server
npm run dev
